Principais formatações

Negrito / Destaque

Nessa oração temos um termo em negrito usando a tag <b> (não semântica).

Nessa oração temos um termo em negrito usando a tag <strong> (semântica).

Itálico / Ênfase

Nessa oração temos um termo em itálico usando a tag <i> (não semântica).

Nessa oração temos um termo em negrito usando a tag <em> (semântica).

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo (não semântica).

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado usando a tag <del>.

Texto Inserido

Podemos marcar um texto como incluído para dar ênfase que ele foi indicado depois, usando a tag <ins>.
Tem também o texto sublinhado que é feito usando a tag <u> (não semântica).

Texto sobrescito

Para inserir texto do tipo x20 + 3, usando a tag <sup>.

Texto subescrito

Para inserir coisas tipo H2O usando a tag <sub>.
